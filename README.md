**Knowledge Orchestration for AI-Assisted Enterprise Presales**

**Overview**

This repository contains the supplementary material accompanying the Master's Capstone research titled:

Knowledge Orchestration for AI-Assisted Enterprise Presales: An Empirical Framework for Context-Aware Response Strategy and AI Readiness

The research investigates how enterprise knowledge can be effectively orchestrated to improve AI-assisted response generation during IT services presales activities. The study combines Design Science Research, Natural Language Processing, Agentic Retrieval-Augmented Generation (RAG), Machine Learning and Inferential Statistics to develop and evaluate an enterprise knowledge orchestration framework.

The repository provides supporting artefacts referenced throughout the research synopsis, including the curated research corpus, annotation guidelines, dataset schema and implementation resources.

**Research Objectives**

The study addresses four research questions:

Determine which characteristics of presales questions require enterprise-specific knowledge.
Identify the enterprise knowledge repositories required across different categories of presales questions.
Evaluate how enterprise knowledge availability influences AI response quality.
Identify organizational knowledge gaps that should be prioritised to improve AI readiness.

Research Methodology

**Details**

The research follows a Design Science Research (DSR) methodology consisting of five stages:

Curated corpus construction
Enterprise knowledge annotation
Agentic RAG implementation
Statistical and machine learning evaluation
Knowledge gap analysis

The primary analytical techniques include:

Binary Logistic Regression
Chi-square Test of Independence
Multiple Linear Regression
Correspondence Analysis
Association Rule Mining
Random Forest
XGBoost
BERT
Dataset

The Curated Enterprise Presales Research Corpus (CEPRC) consists of approximately 900 annotated enterprise presales question–answer pairs collected from publicly available vendor documentation and technical community resources.

Each observation is annotated using a structured framework describing:

Technology Domain
Business Capability
Question Intent
Enterprise Knowledge Required
Response Strategy
Knowledge Gap Category
Response Quality Index (RQI)
